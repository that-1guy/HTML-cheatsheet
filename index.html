The new 'h1' heading about images

'img/' is the tag used to place an image on page

'src' is a required attribute with the image location

'alt' require attribute provides text when image can't be rendered

'width' and 'height' specifies image size in pixel. Without these attributes an image will be rendered in its original size.

Nesting 'img/' tag inside the 'a' tag makes them clickable to their 'URL' source.

We can use the HTML 'figure' which contains 'figcaption' and 'img/' elements.

'figcaption' element specifies the caption on the page

The Audio webpage heading

'audio' element contains a 'source' element

'source' element caontains the source 'src' for the audio

'controls' attribute displays standard playback controls such as play, pause, and volume.

Fallback message is what's diplayed when a user browser doesn't support audio element

'autoplay' element starts audio as soon as page loads

'loop' causes audio to loop continuously

'volume' sets audio defualt volume '0.0 (silent)' to '1.0 (full volume)'

Video Embedding

'video' element contains the 'source' element 'src' and the 'type' element of the video

'controls' attribute displays standard playback controls

'autoplay' element starts audio as soon as page loads

'loop' causes audio to loop continuously

'muted' mutes audio by default

'volume' sets audio defualt volume '0.0 (silent)' to '1.0 (full volume)'

width and height - specifies the width and height of the video player in pixels.
